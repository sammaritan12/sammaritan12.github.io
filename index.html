Machine Learning and NLP

In my final year of university I worked to create software that given a set of texts, was able to identify the likelihood of who the author was. This involved machine learning and natural language processing using libraries such as scikit-learn, NLTK, scipy, and numpy.

PPTXtract

A simple Python script which extracts lyrics from an existing PowerPoint presentation and places them in a new PowerPoint presentation which is more aesthetically pleasing, using desired background images. This was created to extract hundreds of old PowerPoint slides which had poor contrast and images, to a more modern and slick presentation deck.
